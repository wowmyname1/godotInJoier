# Development Roadmap: 3 Months Prototype

**Общая длительность:** 12 недель (3 месяца)
**Длительность спринта:** 1 неделя
**Цель:** Рабочий прототип с полным циклом геймплея.

## Этап 1: Фундамент (Недели 1-3)

### Спринт 1: Базовое движение и камера
- [ ] Реализация контроллера игрока (WASD).
- [ ] Настройка камеры (следование, границы).
- [ ] Базовая физика (коллизии со стенами).
- [ ] Заглушки ассетов (квадрат игрока).

### Спринт 2: Система каста и снаряды
- [ ] Механика зарядки заклинания (нажатие/отпускание).
- [ ] Спавн снарядов (Fireball).
- [ ] Траектория полета и уничтожение снарядов.
- [ ] Базовый хитбокс урона.

### Спринт 3: Система аспектов (Fire & Explosion)
- [ ] Реализация FSM для аспектов.
- [ ] Логика счетчика Over Cast (переход в Explosion).
- [ ] Логика возврата в Fire Aspect.
- [ ] Реализация эффектов: Incomplete Explosion (снаряд), Complete Explosion (взрыв).
- [ ] VFX заглушки (частицы).

## Этап 2: Боевая система и Враги (Недели 4-7)

### Спринт 4: Система здоровья (Dual Health)
- [ ] Компонент `DualHealthComponent`.
- [ ] Логика урона по двум полоскам.
- [ ] Механика регенерации стамины и отсутствия регенерации Max HP.
- [ ] UI: Отрисовка двух полосок и маркера.

### Спринт 5: Враг Crypt (AI)
- [ ] Спрайт/заглушка Crypt.
- [ ] AI: Преследование игрока.
- [ ] AI: Атака (стаггер, отталкивание).
- [ ] Настройка резистов и урона для Crypt.

### Спринт 6: Враг Fire Caster (AI)
- [ ] Спрайт/заглушка Caster.
- [ ] AI: Дистанционная атака (серии по 4 шара).
- [ ] AI: Механика уворота и щита (кулдауны).
- [ ] Баланс взаимодействия игрок <-> Caster.

### Спринт 7: Уровень и окружение
- [ ] Сборка сцены: Комната старта -> Коридор -> Арена.
- [ ] Базовый тайлсет (стены, пол).
- [ ] Расстановка триггеров спавна врагов.
- [ ] Элементы повествования через декор (заглушки).

## Этап 3: Полировка и Геймплейный цикл (Недели 8-12)

### Спринт 8: Боевой цикл и баланс
- [ ] Интеграция всех врагов на арене.
- [ ] Настройка волн спавна.
- [ ] Первичный баланс урона и здоровья.
- [ ] Добавление звуковых эффектов (SFX) для выстрелов и ударов.

### Спринт 9: Чувственность (Game Feel)
- [ ] Тряска камеры (Screen Shake) при взрывах.
- [ ] Отдача при касте.
- [ ] Анимации "hit stop" при попаданиях.
- [ ] Улучшение партиклов (VFX).

### Спринт 10: Narrative & Atmosphere
- [ ] Детализация окружения (декорации).
- [ ] Освещение и тени (Godot 2D Lighting).
- [ ] Проверка читаемости истории через левел-дизайн.

### Спринт 11: Финальный баланс и Багфикс
- [ ] Тестирование сложности (Crypt + Caster + Boss concept).
- [ ] Исправление критических багов.
- [ ] Оптимизация производительности.

### Спринт 12: Релиз прототипа
- [ ] Сборка версии для команды.
- [ ] Написание краткого отчета о тестах.
- [ ] Планирование следующего этапа на основе фидбека.
- [ ] Добавление фоновой музыки (опционально).
